Conceptos Web

Una página web es un documento digital accesible en internet que contiene texto, imagenes y otros elementos interactivos.

    HTTP (Protocolo de transparencia de hipertexto)

    Es como un idioma que usan las computadoras para comunicarse entre sí y poder mostrarte las páginas web que buscas.

    WWW (Word Wide Web)

    Es un sistema de distribución de documentos y recursos a través de internet que permite a los usuarios acceder a páginas web y otros contenidos

    Dominio

    Elementos Web

    HTML (Hipertext markup language)

    Facilitar el intercambio de información mediante la creación de una plataforma basada en el hipertexto.

    Hipertexto

    Nació para presentar información para saltar de un lado a otro de forma no secuencial.

    CSS

    Facilitar el diseño visual sin alterar la estructura básica definida de HTML.

    UX

    Es como se siente una persona al interactuar con un producto, servicio o sistema.

    Prototipado

    Es crear un boceto o una versión preliminar de una página web antes de desarrollarla completamente.

    Interactividad

    La interactividad en el desarrollo web se refiere a la capacidad de un sitio web de responder a las acciones de un usuario
